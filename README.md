## ★ 프로젝트 소개
- C# WinForms을 기반으로 MDI 기반 일정 관리 프로그램을 구현함.
- 회사 일정관리용 캘린더를 컨셉으로 하여 각 팀별 그룹일정을 공유할 수 있게 설계함.
- 사용자 로그인 시 캘린더 UI가 표시되도록 구성하였으며, MySQL과 연동하여 저장된 사용자 일정을 UI에 시각적으로 표시하였음.
- 사용자가 UI에서 특정 날짜를 선택하여 일정을 직접 추가 및 삭제할 수 있는 기능을 구현함. 부가적으로 날짜 검색 기능을 통해 검색 시 해당 날짜가 포함된 달력으로 이동하도록 구현함.
- 사용자 환경설정 기능으로 기본 테마, 계절별 자동 테마 변경 기능을 구현하였으며, 단축키 기능을 추가하여 프로그램 사용 편의성을 강화함.

## ★ 담당 역할
### C# WPF GUI 프로그램 구현
- C# WPF 관리자 프로그램 환경을 MVVM구조로 구현하여 유지보수 및 확장성을 확보
- 실제 공정에서 중요한 시각화(LiveCharts2, OpenCV) 및 실시간성(센서 데이터, Log, 물류 처리량)을 고려하여 UI에 반영
- Flask 서버와 SQL Lite와 연동하여 검수 기록 및 배송 상태 관리 등 데이터 수집

### YOLOv8을 통한 학습 모델 생성
- 모델 학습을 위해 Roboflow 활용 라벨링 및 데이터셋 생성 
- 이미지를 정규화 및 다양화하여 과적합 방지, 정밀도 향상, 오검출 감소 효과 강화
- 모델 학습(정확도 98.6%)결과 그래프 시각화 및 PyTorch로 학습한 .pt 파일을 .onnx 파일로 변환 

### WPF 기반 ONNX 불량 검출
- WPF GUI환경에 .NuGet ONNX Runtime을 설치 및 적용하여 프레임워크 유연성 확보 및 성능 최적화
- ONNX파일과 WPF의 직접 연동을 통한 불량 검출 딜레이 감소

### OpenCV 활용 이미지 바운딩 박스 시각화
- OpenCV를 활용하여 검출된 영역을 물류 이미지에 바운더리 박스로 시각화
- 시각화한 이미지 검출 기록을 수동으로 확인해야 할 경우 활용할 수 있도록 .exe 실행경로 폴더에 이미지 자동 저장
